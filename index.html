2 Formulating the Problem

Formulating a linear programming problem is like writing a recipe for success! We need three key ingredients:

Decision Variables

These are the things we can change, like how many items to produce.

x₁ = tables to produce
x₂ = chairs to produce

Objective Function

This is what we want to maximize or minimize, like profit.

Maximize Z = 70x₁ + 50x₂

Constraints

These are the limits we have to work within.

3x₁ + 2x₂ ≤ 120 (wood hours)
1x₁ + 2x₂ ≤ 80 (finishing hours)
x₁ ≥ 0, x₂ ≥ 0

Furniture Manufacturing

A furniture company makes tables and chairs. Each table requires 3 hours of woodworking and 1 hour of finishing. Each chair requires 2 hours of woodworking and 2 hours of finishing. The company has 120 hours of woodworking time and 80 hours of finishing time available. Each table sells for $70 profit and each chair for $50 profit. How many of each should be made to maximize profit?

3 Graphical Solution Method

For problems with two variables, we can draw the solution! Follow these steps:

1

Plot the Constraints

Turn each inequality into a line on the graph.

2x₁ + x₂ ≤ 10
x₁ + 3x₂ ≤ 15
2

Find the Feasible Region

This is the area where all constraints are satisfied.

3

Plot the Objective Function

Draw lines representing different values of Z.

Z = 5x₁ + 3x₂
4

Find the Optimal Point

The best solution is always at a corner of the feasible region!

Key Insight

You can imagine the objective function as a line that you "push" outward until it just touches the feasible region!

4 Converting to Standard Form

Before using the Simplex Method, we need to put our problem in standard form with these rules:

All constraints are equalities (using slack/surplus variables)

All variables are non-negative

Objective function is in maximization form

Conversion Techniques

Click the steps to see how we transform the problem!

Original Problem

Minimize W = 4x₁ - 2x₂ + 3x₃
Subject to:
x₁ + 5x₂ - x₃ ≥ 12
2x₁ - x₂ + 4x₃ ≤ 18
x₁ ≥ 0, x₂ unrestricted, x₃ ≥ 0

After Step 2: Inequality Conversion

Minimize W = 4x₁ - 2x₂ + 3x₃
Subject to:
x₁ + 5x₂ - x₃ - s₁ = 12
2x₁ - x₂ + 4x₃ + s₂ = 18
x₁ ≥ 0, x₂ unrestricted, x₃ ≥ 0, s₁, s₂ ≥ 0

After Step 3: Unrestricted Variables

Minimize W = 4x₁ - 2(x₂⁺ - x₂⁻) + 3x₃
Subject to:
x₁ + 5(x₂⁺ - x₂⁻) - x₃ - s₁ = 12
2x₁ - (x₂⁺ - x₂⁻) + 4x₃ + s₂ = 18
x₁ ≥ 0, x₂⁺ ≥ 0, x₂⁻ ≥ 0, x₃ ≥ 0, s₁, s₂ ≥ 0

After Step 4: Objective Conversion

Maximize -W = -4x₁ + 2(x₂⁺ - x₂⁻) - 3x₃
Subject to:
x₁ + 5(x₂⁺ - x₂⁻) - x₃ - s₁ = 12
2x₁ - (x₂⁺ - x₂⁻) + 4x₃ + s₂ = 18
x₁ ≥ 0, x₂⁺ ≥ 0, x₂⁻ ≥ 0, x₃ ≥ 0, s₁, s₂ ≥ 0

Final Standard Form

Maximize Z = -4x₁ + 2x₂⁺ - 2x₂⁻ - 3x₃
Subject to:
x₁ + 5x₂⁺ - 5x₂⁻ - x₃ - s₁ = 12
2x₁ - x₂⁺ + x₂⁻ + 4x₃ + s₂ = 18
x₁, x₂⁺, x₂⁻, x₃, s₁, s₂ ≥ 0

5 The Simplex Method

The Simplex Method is like a mountain climber who moves from one peak to another, always going higher until reaching the top!

1

Start at a Corner

Begin with a basic feasible solution (usually using slack variables).

2

Choose an Entering Variable

Pick a variable that will improve the objective function.

3

Choose a Leaving Variable

Use the minimum ratio test to maintain feasibility.

4

Pivot

Update the tableau to move to a new basic feasible solution.

5

Check for Optimality

Repeat until no more improvement is possible.

Key Insight

Each pivot moves us to a better corner point of the feasible region!

6 The Duality Concept

Every linear programming problem has a shadow twin called its dual problem. They're like two sides of the same coin!

Primal ↔ Dual Relationship

  • Maximization ↔ Minimization
  • Constraints ↔ Variables
  • Objective coefficients ↔ RHS values

Duality Theorems

  • Weak Duality: Dual objective ≤ Primal objective
  • Strong Duality: At optimality, Dual = Primal
  • Complementary Slackness: Tells us which constraints are binding

Practical Uses

  • Finding shadow prices (value of resources)
  • Sensitivity analysis (what-if scenarios)
  • Alternative solution method

Key Insight

The dual variables tell us how much the objective function would change if we had one more unit of each resource!

Primal Problem

Maximize Z = 3x₁ + 2x₂
Subject to:
2x₁ + x₂ ≤ 18
x₁ + 3x₂ ≤ 15
x₁, x₂ ≥ 0

Dual Problem

Minimize W = 18y₁ + 15y₂
Subject to:
2y₁ + y₂ ≥ 3
y₁ + 3y₂ ≥ 2
y₁, y₂ ≥ 0

Optimal Solutions

Primal

x₁ = 6, x₂ = 3

Z = 24

Dual

y₁ = 1.2, y₂ = 0.2

W = 24

Interpretation

The dual variables (y₁ = 1.2, y₂ = 0.2) tell us that:

  • One more unit of resource 1 is worth $1.20
  • One more unit of resource 2 is worth $0.20
